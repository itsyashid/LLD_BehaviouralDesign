**Benefits of Behavioural Design pattern:**

1.Scalability and Maintainability
2.Encapsulation of Object creation
3.Improved Reliability.

**Implementing the below Problem Statement keeping in mind the different design patterns
that come inside Behavioural Design (mainly Strategy and Observer Design pattern)**

**ProblemStatement**

Develop a text editor that allows user to input text and apply different formatting styles 
(e.g. lowercase, uppercase etc.). Also, there should be a system waiting for the text to be updated.
Everytime the text is updated the text editor should notify the observers.
Also text formatting operations should be encapsulated and execute them on the editors text.
